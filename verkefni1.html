Figuring out how to get started

I started by reading the websites Atli Tobiasson and Sæmundur Guðmundsson made last year. Their websites gave me an overview of how I should do things. Then, to get more detailed instructions I searched how to make a portfolio website on youtoube. This is the video that helped me the most to get started.

Building the website

After I finished the video above I went strait to the website HTML5 UP! and explored templates. I finally found a template that fitted well for this project. It was the Editorial template. I downloaded that template and opened it in Visual studio Code. Before I could start to modify the template I had to download live extension to Visual Studio to be able to see changes to the website live.

Editing text

To find out how to change text, I used Ctrl + F and searched for text I saw on the websita and found the corresponding text in the html file. I found out relatively quickly that <p>Text </p> was the syntax for paragraph and <hi>Header </hi> was the syntax for headers where i stands for 1 to 4 and represents different headings.

Then I found out that <body>Text </body> was used to split text on pages to different bodies and <section>Text </section> to split the text to different sections. Those bodies and sections could then be styled using CSS which I will explan in the next chapter. To display different stylings in yeach section <div>Text </div> was used to divide the section even more.

Styling with CSS

It took me quite long time to find out how to change the styling using CSS worked. Googling did not yield me satisfactory results about how the structure of the CSS code worked but I finally was able to figure it out by myself using Ctrl+F and reading the comments in the CSS script. The css script was split into different id´s which then contanied many classes. The id´s where then called in the html-file that were to by styled like this: <body id ="some id" >. Then if a sector or div were to by style further more "class = "some class"" was added to it like this <sector id ="some id" >. An examle of css id and class is seen in the picture below.

In the image above, the id´s name is image which holds the information for how all images in a body with specifi id look. Then if images within that body need to be different that is done with a class. Here we have general class namend image, then there is image right and left class also.
